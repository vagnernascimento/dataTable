Module dataTable.repository

This module generates a compatible Json DataTable repository.

Release: $Id: repository.lua,v1.0 2009/02/19 Vagner Nascimento Exp $

[obj]:get ()
Returns a string with the result in Json
[obj]:new (itens, arg, recordName)
Creates a new instance of dataTable repository

Parameters

  • itens: Lua table with itens for dataTable
  • arg: Accepts a lua table with these arguments: startIndex, results, sort and dir
  • recordName: String with the name of the Json resulting list

Usage:

 obj = require("dataTable.repository")
 ex1.: obj:new(table_itens)
 ex2.: other_obj:new(table_itens,{startIndex=0,results=30,sort='field1'},'results')
[obj]:response (headerType, headerSubtype, headerCharset)
Renders the instance of dataTable repository

Parameters

  • headerType: String with the resulting HTML type header. The default value is 'application'.
  • headerSubtype: String with the resulting HTML subtype header. The default value is 'json'.
  • headerCharset: String with the resulting HTML charset. The default value is 'ISO-8859-1'.

Usage:

 ex1.: obj:response()
 ex2.: obj:response('text','plain','UTF-8')
